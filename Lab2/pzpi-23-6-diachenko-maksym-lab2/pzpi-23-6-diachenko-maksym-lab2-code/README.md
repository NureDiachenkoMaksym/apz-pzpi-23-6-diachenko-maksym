# Lab2 Mobile Application

This directory contains a demonstration Android/Kotlin mobile application for the Natural Resources Analysis Software System.

The application is designed as a mobile client for field operators and analysts. It supports viewing natural resource indicators, creating local measurement drafts, and synchronizing them with the backend API.

Main technologies:
- Kotlin
- Android
- Jetpack Compose
- ViewModel
- Repository pattern
- Ktor client
- JSON API

The source code is educational and demonstrates the architecture required for Lab2.
