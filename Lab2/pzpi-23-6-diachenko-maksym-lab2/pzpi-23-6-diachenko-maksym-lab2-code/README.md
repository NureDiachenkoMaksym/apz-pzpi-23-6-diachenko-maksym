# Mobile client — Natural Resources Analysis

Android Kotlin mobile client for field observations and quick analytics.

## Main features

- View natural resource dashboard.
- Create field observations.
- Synchronize data with backend REST API.
- Support Ukrainian interface texts.
- Demonstrate layered mobile architecture.

## Main packages

- `data.model` — data classes.
- `data.network` — API contract.
- `data.repository` — data access abstraction.
- `ui` — Compose UI.
- `viewmodel` — UI state and business actions.
