# Lab3 Web Application - Natural Resources Analysis

This directory contains a lightweight front-end prototype for the Natural Resources Analysis Software System.

## Features
- user and administrator modes;
- Ukrainian and English localization;
- localized date/time formatting;
- locale-aware sorting of resource names;
- resource dashboard, filters and analytical table;
- admin panel for users, reference data, backups, import/export;
- mock REST API layer for integration with backend services.

## Run
Open `index.html` in a browser or run a simple local server:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.
