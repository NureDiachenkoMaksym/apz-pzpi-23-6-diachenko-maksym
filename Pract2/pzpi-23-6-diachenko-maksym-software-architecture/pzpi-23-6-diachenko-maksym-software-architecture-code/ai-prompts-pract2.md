1. Поясни архітектуру Google Earth Engine українською мовою.
2. Опиши клієнт-серверну взаємодію, API-рівень, каталог даних і серверне виконання обчислень.
3. Порівняй Google Earth Engine із типовою вебсистемою аналізу природних ресурсів.
4. Підготуй приклад JavaScript-коду для обчислення NDVI.
5. Опиши переваги та обмеження хмарної архітектури для геопросторової аналітики.
