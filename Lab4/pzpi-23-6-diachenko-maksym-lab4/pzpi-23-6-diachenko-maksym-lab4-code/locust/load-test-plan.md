# Load test plan

Мета: перевірити, як зростає кількість оброблених запитів за секунду при збільшенні кількості реплік API.

Сценарії:
1. Відкрити список ресурсів.
2. Виконати фільтрацію за типом ресурсу.
3. Отримати короткий підсумковий звіт.
4. Перевірити health endpoint.

Метрики:
- RPS;
- p95 latency;
- error rate;
- CPU and memory usage.
