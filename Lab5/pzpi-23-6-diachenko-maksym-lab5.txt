Презентація готового проєкту

Назва проєкту: EcoResource Analytics
Тема: програмна система моніторингу та аналізу природних ресурсів.

Короткий опис проєкту
EcoResource Analytics - це інтегрована програмна система для збору, зберігання, перевірки, аналітичної обробки та представлення даних про природні ресурси. Система охоплює земельні, водні та лісові ресурси, підтримує REST API, базу даних PostgreSQL, аналітичні модулі, адміністративні функції, IoT-клієнт та web-інтерфейс.

Заплановані та виконані функції
1. Серверна частина REST API - виконано.
2. База даних PostgreSQL зі структурованими довідниками та вимірюваннями - виконано.
3. Авторизація JWT та рольова модель доступу - виконано.
4. Аналітика trend/summary/anomalies - виконано.
5. Адміністративні функції: користувачі, quality_status, audit log - виконано.
6. IoT-клієнт для передачі вимірювань - виконано.
7. Web dashboard з фільтрами, локалізацією та admin-режимом - виконано.
8. Docker Compose, Kubernetes HPA та Locust для демонстрації масштабування - підготовлено.
9. Реальні IoT-пристрої, карти та AI-прогнозування - заплановано для майбутнього розвитку.

Матеріали представлення проєкту
Для демонстрації використовується web dashboard, структура проєкту, backend API, IoT-клієнт, база даних, Docker/Kubernetes/Locust файли та сценарій demo-video.
